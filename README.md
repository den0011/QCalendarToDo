# QCalendarToDo

Настольный ежедневник и планировщик задач на Qt5/Qt6 с поддержкой тем оформления, статистики и поиска.

## Возможности

### Планировщик дня
- Таблица задач с интервалами по **15 минут** (96 строк на весь день)
- Три поля на каждый интервал: **Заголовок**, **Сообщение**, **Описание**
- Визуальное разделение: прошедшие интервалы подсвечиваются, будущие — чистые
- **Линия текущего времени** — горизонтальная полоса, которая всегда указывает на текущий момент
- Автопрокрутка к текущему времени при открытии
- Контекстное меню по правой кнопке: очистить строку, скопировать в буфер обмена
- Заметки на выбранный день в отдельной вкладке

### Календарь
- Встроенный календарь с навигацией по месяцам (кнопки ◀ ▶ и «Сегодня»)
- Выделение сегодняшнего числа акцентным цветом
- Подсветка выходных дней
- Индикаторы (точки) на днях, в которых есть записи

### Статистика
- **Круговая диаграмма** выполненных и предстоящих интервалов текущего дня
- Панель итогов дня: количество завершённых и оставшихся интервалов, задачи дня
- Диалог расширенной статистики:
  - Общее число задач, дней с записями, заметок, размер базы данных
  - График задач по месяцам (последние 6 месяцев)
  - Распределение задач по дням недели
  - Отчёт за произвольный период с экспортом в HTML
  - Анализ продуктивности с рекомендациями

### Поиск
- Полнотекстовый поиск по всем записям базы данных
- Фильтры по полям: заголовок, сообщение, описание, заметки
- Учёт регистра опционально
- Двойной клик по результату — переход на нужную дату

### Настройки
- Выбор цветов для прошедших и предстоящих интервалов
- Настройка шрифтов для задач и заметок
- Уведомления в системном трее при наступлении времени задачи

### Темы оформления
Четыре встроенные темы:

| Тема | Описание |
|------|----------|
| Light | Светлая классическая |
| Dark | Тёмная |
| Blue | Синяя |
| Green | Зелёная |

## Требования

- **Qt** 5.9+ или Qt 6.x
- Модули Qt: `core`, `gui`, `widgets`, `charts`, `sql`
- Компилятор с поддержкой C++11
- SQLite (входит в состав Qt)

## Сборка

```bash
git clone https://github.com/your-username/QCalendarToDo.git
cd QCalendarToDo
qmake QCalendarToDo.pro
make          # или mingw32-make на Windows
```

Либо откройте `QCalendarToDo.pro` в **Qt Creator** и нажмите Build.

## Хранение данных

Все данные хранятся в SQLite-базе `my_database.db` в рабочей директории приложения.
Настройки сохраняются в конфигурационный файл `settings.ini`.

Структура базы данных:
- `Dates` — дни с записями
- `Times` — задачи (привязаны к дате и времени)
- `Strings` — заметки на день
